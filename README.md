
# Getting Started

## Step 1: Install Node Modules

Execute command line:

```bash
# using npm
npm install

```

## Step 2: Start Application

Execute command line:

```bash
# using npm
npm start

```
### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
